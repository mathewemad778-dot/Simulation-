# Simulation-
This project started with a simple question: what if you could simulate your life decisions before actually living them? What if small daily habits like sleeping one hour less, skipping the gym, studying a bit more, or taking higher financial risks could be projected into a 10-year future?

The idea was not to build just another calculator or random number generator. The goal was to create a dynamic life simulation engine that reacts to user input and evolves year after year. Instead of showing static results, the system models progression. Each year depends on the previous one. Wealth grows or shrinks based on income, investment decisions, and risk tolerance. Health responds to sleep patterns and gym habits. Mental state shifts according to balance and pressure. Career evolves through study consistency and long-term discipline.

The experience begins with a clean interactive form where the user defines their current state: age, income, sleep hours, study commitment, lifestyle habits, and risk level. Once the simulation runs, the app generates a 10-year timeline. Each year appears as a visual card displaying Wealth, Health, Mental, and Career scores. The design follows a SaaS-inspired two-column layout, separating user input from results for clarity and usability.

The real impact comes at the end. After the ten simulated years, the system analyzes the final state and calculates an overall life score. Instead of leaving users with raw numbers, it provides a meaningful evaluation — whether their trajectory is excellent, balanced, risky, or collapsing. This transforms the project from a simple visualization tool into a reflective decision-making experience.

From a technical perspective, the application is built with vanilla HTML, CSS, and JavaScript. The simulation logic is modular and structured to keep yearly progression consistent and scalable. The UI uses CSS Grid for layout and custom components for interactive elements. Theme switching is handled through CSS variables, allowing smooth transitions between dark and light modes without rewriting styles.

This project represents more than UI styling or basic DOM manipulation. It demonstrates state management across iterations, dynamic rendering, user-driven simulations, conditional logic systems, responsive layout design, and theme architecture using design tokens. It combines logic and interface in a way that makes the data feel alive.

Ultimately, this app is an experiment in visualizing consequences. It turns abstract habits into visible long-term outcomes. It’s not about predicting the future with certainty, but about understanding how consistent decisions compound over time.

And that’s where the real value lives — in making the invisible effects of daily choices visible
